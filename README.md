# 🌤️ App básica de clima con Spring Boot + Open‑Meteo

Aplicación sencilla en **Spring Boot** que expone un endpoint REST `/weather` para consultar el clima actual de cualquier ciudad usando coordenadas (latitud y longitud). La aplicacióñ consume la API pública de **Open‑Meteo**, que no requiere registro ni API key.

---

## 📌 Requisitos previos
- **Java 17 o superior**
- **IntelliJ**
- **Postman**

---

## 🚀 Instalación y ejecución

1. **Clonar repositorio**     

2. **Importar en IntelliJ**  

3. **Ejecutar como aplicación Java**

4. **Usar GET en Postman la siguiente URL http://localhost:8080/weather?lat=19.43&lon=-99.13**
